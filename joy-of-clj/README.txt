With Emacs and isend-mode

https://github.com/ffevotte/isend-mode.el

1. <F12> shortcut key (or C-x 2 to split)
2. M-x shell (bottom)
3. java -cp clojure/clojure-1.6.0.jar clojure.main
4. Open a .clj file (top)
5. M-x lisp-mode (top)
6. M-x isend-associate RET (top)
7. Type shell, TAB, RET
8. Add a comment line (can be anything) as the first block
9. Use shortcut keys S-RET (block) and M-RET (buffer) to send
